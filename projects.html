Tenant Management App

A GUI that displays tenant, apartment, and rent information. The user can select a tenant / apartment from the combo box at the upper left, and any notes about the tenant / apartment are displayed in the text box below. Key details about the tenant and apartment are displayed in the center, and rent payments can be applied using the fields to the right. The user can also charge all current tenants rent. Developed in C# and WPF Forms using Visual Studio 2022. (Winter 2023.)
